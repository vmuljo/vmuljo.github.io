MindUNO - An IoT System to Collect and Visualize Brianwave Data

The purpose of our IoT system, “MindUno”, is to collect raw electroencephalography brainwave data, locally parse and process it on an ArduinoUno, and communicate that data to a remote node (our laptop) where we can visualize the data in a meaningful, human-understandable way. We use a RaspberryPi to transmit the data to our laptop/virtual machine over MQTT.

Check out our GitHub page!

Temperature Monitor - An Embedded System to Detect Changes in Temperature

The purpose of my embeeded system is to sense the environmental temperature and alert the user with LED lights and noise when the temperature surpasses a set threshold. The threshold can be set by the user using a rotary encoder that has been coded and processed on an ArdunoUno to set proper temperature ranges. The user may also set a certain frequency as their noise of choice by clicking a button that allows the rotary encoder to switch from setting the temperature threshold to setting the frequency.
